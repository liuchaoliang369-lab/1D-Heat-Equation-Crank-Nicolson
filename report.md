\# Crank–Nicolson 求解 1D 热方程 — 实现与验证



\*\*作者\*\*：朝亮 刘  

\*\*日期\*\*：2026-03 (可改为实际完成日期)



\## 摘要

本文实现了 1D 热方程 \\(u\_t = u\_{xx}\\) 的 Crank–Nicolson 数值格式并使用 Thomas 三对角求解器求解线性系统。通过与解析解比较，我们验证了算法在空间与时间上均呈二阶收敛性；并给出时间收敛区间内的拟合阶数与示例运行误差。代码与图像已上传于仓库（见 `main.py`、`result.png`、`convergence.png`、`timetest.png`）。



\## 问题描述

求解如下初边值问题：

\\\[

u\_t = u\_{xx},\\quad x\\in(0,1),\\ t>0,

\\]

边界条件：\\(u(0,t)=u(1,t)=0\\)，初值取

\\\[

u(x,0)=\\sin(\\pi x).

\\]

解析解为

\\\[

u(x,t)=\\sin(\\pi x)e^{-\\pi^2 t}.

\\]



\## 数值方法

空间采用中心差分（二阶）离散 \\(\\partial\_{xx} u\\)，时间采用 Crank–Nicolson（时间二阶、隐式）。对内部网格节点，离散后在每一时间步需解三对角线性系统，本文用 Thomas 算法（O(N)）高效求解。



离散形式（给出内部节点）：

\\\[

\\frac{u^{n+1}\_i - u^n\_i}{\\Delta t} = \\frac{1}{2}\\left(\\delta\_{xx} u^{n+1}\_i + \\delta\_{xx} u^n\_i\\right).

\\]



\## 实现细节

\* 线性求解：自实现 Thomas（三对角求解器）。

\## 实验设置

\* 空间网格：在收敛试验中取 \\(N = 51,101,201,401\\) 节点（对应 \\(h=1/(N-1)\\)）。  

\* 时间步：用于时间收敛试验时固定空间为较细的网格（以避免空间误差污染），并采用一系列 \\(\\Delta t\\)（见下表）。  

\* 终止时间：本文示例取 \\(T=0.1\\)。



\## 主要结果（数值）

\*\*示例运行（nx=101）\*\*  

\- L2 error ≈ `2.138979095998931e-05`  

\- max error ≈ `3.024973247189866e-05`



\*\*空间收敛\*\*  

使用不同网格的 L2 误差拟合 log-log 直线，估计空间收敛阶（L2）≈ \*\*1.99984\*\*（接近理论二阶）。参见 `convergence.png`。



\*\*时间收敛（示例数据）\*\*  

对固定较细空间网格（使空间误差远小于时间误差）的多组 \\(\\Delta t\\) 计算 L2 误差，部分数据如下：



| Δt | L2 error |

|---:|:---------|

| 1.000e-02 | 2.10025e-04 |

| 5.000e-03 | 5.14615e-05 |

| 2.000e-03 | 7.10883e-06 |

| 1.000e-03 | 7.74342e-07 |

| 5.000e-04 | 8.09216e-07 |

| 2.500e-04 | 1.20510e-06 |

| 1.250e-04 | 1.30407e-06 |

| 6.250e-05 | 1.32882e-06 |



对不受空间误差主导的前几组数据进行拟合得到时间收敛阶 \*\*≈ 2.0290\*\*（参见 `timetest.png`）。当 Δt 继续减小时，误差趋于平台期或略有上升，原因是空间误差和舍入误差开始占主导。



\## 讨论

\* 在适当的（非污染）区间内，Crank–Nicolson 在时间上呈二阶收敛，与理论一致。  

\* 当 Δt 足够小而空间分辨率固定时，总误差受空间离散误差主导，故误差不再随 Δt 降低（这也是实验中出现负的局部估计阶的原因）。  

\* Thomas 算法在运行时间上表现出接近线性的增长（见 `runtime.png`，若已生成）。
